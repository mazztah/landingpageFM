// Quickstart

Deployment auf Google Cloud Run

bash 
# 1. Repository auf GitHub hochladen
git add .
git commit -m "Initial commit"
git push origin main

# 2. In Google Cloud Console → Cloud Run → "Create Service"
#    - Source Repository: dein GitHub Repo
#    - Branch: main
#    - Dockerfile: automatisch erkannt

# 3. Wichtige Umgebungsvariablen setzen:
TELEGRAM_BOT_TOKEN=...
GROQ_API_KEY=...
SUPABASE_URL=...
SUPABASE_KEY=...
ANTHROPIC_API_KEY=...   # optional
DASHSCOPE_API_KEY=...   # optional für Video
GEMINI_API_KEY=...      # optional für Lyria

# 4. Deploy starten
# Cloud Run erkennt das Dockerfile automatisch und deployed.
